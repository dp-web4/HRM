## Cross-Track Coupling Analysis

### Original Hypothesis (from 12:35 check)

**Claim**: Training track D5/D9 improvements support primary track baseline

**Evidence Timeline**:
```
09:01 - T023: 75% success (D5/D9 ≈ 0.70/0.72)
12:02 - Session 20: Partial recovery (D5/D9 = 0.45/0.50)
15:00 - T024: 50% regression (D5/D9 estimated ≈ 0.55/0.50)
```

### Hypothesis Status: **WEAKENED**

**Problems with coupling hypothesis**:

1. **Directionality contradiction**:
   - T023 high → Session 20 recovery: Supports positive coupling
   - Session 20 recovery → T024 regression: Suggests negative coupling OR no coupling

2. **Lag inconsistency**:
   - If 3-hour lag: Session 20 recovery should predict T024 improvement
   - Instead: T024 regressed 3 hours after Session 20 recovered

3. **Simpler explanation available**:
   - Both tracks oscillating independently at threshold boundaries
   - Common bistable dynamics (shared mechanism)
   - No direct inter-track influence

### Revised Hypothesis: **Independent Bistable Oscillation**

**Claim**: Training and primary tracks show parallel bistable dynamics without strong coupling

**Supporting Evidence**:
- Both tracks oscillate at boundary (not stable above threshold)
- Both tracks show bistable state switching (confabulation/hedging, partnership/educational)
- Curriculum maintains boundary oscillation but doesn't push above threshold
- No consistent directional coupling observed

**Mechanism**: Shared underlying bistable architecture
- Same model weights
- Same thermodynamic preferences (educational default favored)
- Same curriculum limitations (insufficient for stable high state)
- **But**: Independent state trajectories per track
