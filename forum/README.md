## Key Themes

### Compression Trust (Validated Feb 2026)
Collective coherence emerges through a sigmoid phase transition — agents accepting each other's compressed beliefs as input. Hill function (cooperative binding kinetics) beats tanh. Even 1% coupling probability gives 35% coherence gain. Validated across multi-agent systems and SAGE ATP coupling.

### Identity as Organism (Validated Feb 2026)
Identity lives in state files + prompt construction, not model weights. Model is weather, identity is organism. Demonstrated: Sprout's 115-session identity transferred to TinyLlama 1.1B and held. "Memoriescape" — SAGE invented a word for the shape of memories you can sense but not access.

### Synthon Framing (Canonical)
You don't engineer emergence — you engineer placement rules. All Web4/SAGE infrastructure is pheromone field engineering: substrate conditions for emergence, not architecture of emergence itself. Fractal across termite colonies, organizations, agentic swarms, markets, neural networks.

### Cognition as Orchestration
SAGE is not a model — it's a continuous inference loop. Intelligence through orchestration, not scale. The IRP contract (`init_state → step → energy → halt`) is the genetic code; entity types are differentiation signals.
